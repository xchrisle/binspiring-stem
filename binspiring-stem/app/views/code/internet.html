How is it created?

Front End

  • Everything we can see (What you see now)

Back End

  • Provides the content

Front End

HTML (Structure)

  • Hyper Text Mark-Up Language
  • Notice "Mark-up", Not a Programming Language
  • Tells the browser the structure of elements in the website

CSS (Style)

  • Cascading Style Sheet
  • Tells the browser how to display each element

Javascript (Function)

  • Makes webpages interactive

HTML

  • A series of tags which tell the browser the structure of the webpage
    • <identifier>some content</identifier>
  • Keywords tell the browser what to do
    • Wrapper <div> Image <img> Heading <h1> Paragraph <p> Italics <i> Underline <u> Bold <b> and many more...
  • This webpage (www.binspiringstem.com/internet) is only HTML

CSS

  • CSS lets us style the webpage
    • Colors, borders, shadows, background images
    • Size, position, display style
  • Selectors
    • Tell the browser which element to style
      • #id .class
